Holbegram Project

Holbegram is a Flutter mobile application inspired by Instagram social media app.
This project is part of the Holberton School curriculum and focuses on learning Flutter, Dart, Firebase, and mobile UI development.

Project Objectives:

- Understand Flutter project structure

- Build UI screens using Flutter widgets

- Navigate between screens

- Prepare Firebase integration

- Follow a given design specification
